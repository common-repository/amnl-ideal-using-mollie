Add iDeal support to WooCommerce using Mollie.

== Description ==
*Please note that this plug-in is still a __work in progress__!*

Add iDeal support to WooCommerce using Mollie as a payment service provider.
Requires PHP 5.3+ (uses namespaces and spl_autoload_register).

== Installation ==

Add the plug-in to your WordPress installation and go to your WooCommerce settings.
The plug-in should be listed under the Payment Gateways.
There you'll be able to enable the payment method and set your Mollie account information.
